# Changelog

## 0.1.0

- Added deterministic JSON-in/JSON-out floor plan generation engine and CLI.
- Added strict input parsing, validation dry-run mode, and machine-readable diagnostics.
- Added published input/output JSON Schema artifacts with schema version `1.2`.
- Added stable adapter-facing `externalId` values and Rhino/Grasshopper layer names.
- Added a local web workbench for sample loading, JSON editing, validation, generation, SVG preview/export, and output export.
- Added portable hypergraph output with `DataNode`-compatible JSON, explicit nodes, multi-member hyperedges, incidence records, and matrices.
- Added representative samples and golden contract tests for rectangular, L-shaped, moderately irregular, and infeasible floorplates.
